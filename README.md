## Project Description

The Temperature Converter is a Java-based application that allows users to convert temperature values between different units such as Celsius, Fahrenheit, and Kelvin. The application provides a simple and user-friendly interface for performing accurate temperature conversions instantly.

---

## Project Scope

The scope of this project is to develop a temperature conversion tool that helps users convert temperature values among various scales efficiently. The project demonstrates the use of Java programming concepts, user input handling, mathematical calculations, and graphical user interface development.

---

## Features

* Celsius to Fahrenheit Conversion
* Fahrenheit to Celsius Conversion
* Celsius to Kelvin Conversion
* Kelvin to Celsius Conversion
* Fahrenheit to Kelvin Conversion
* Kelvin to Fahrenheit Conversion
* User-Friendly Interface
* Instant and Accurate Results
* Input Validation

---

## How to Run

1. Install Java JDK.
2. Compile the program:

   ```
   javac TemperatureConverter.java
   ```
3. Run the application:

   ```
   java TemperatureConverter
   ```
4. Enter the temperature value, select the conversion type, and click the Convert button.

---

## Sample Conversions

* 0°C → 32°F
* 100°C → 212°F
* 25°C → 298.15 K
* 273.15 K → 0°C

The application displays the converted temperature value instantly based on the selected conversion type.
